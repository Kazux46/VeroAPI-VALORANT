# VeroAPI
A simple Valorant profile and skin customizer that uses the local Riot Client API.
## Features
- Change displayed rank and leaderboard position
- Update player card, title, and level border
- Equip weapon skins
- Auto-reapply presence feature
## Setup
1. Install dependencies:
npm install
2. Start the application:
npm start
3. Ensure Valorant is running before clicking "Authorize".
## Tech
- Node.js / Express
- Vanilla JS, HTML, CSS
